# 更新日志

## v2.1.0 - 2026-07-03

### 新增

- 新增首页，可选择进入 `TLM 计算` 或 `计时器`。
- 新增计时器页面：
  - 秒级倒计时，支持直接输入几百秒并显示剩余秒数与分钟/秒。
  - 预设倒计时：3 分钟、5 分钟、10 分钟、14 分钟。
  - 自定义分钟倒计时。
  - 倒计时备注显示。
  - 正计时器，支持开始、暂停、重置。
- 新增 GitHub Actions 自动打包流程，支持下载 Android APK 和 Windows 包。

### 改进

- TLM 计算页面改为通过预设管理间距、TLM 数量、通道宽度和测试电压。
- 主界面只保留预设选择、保存名称和电流输入，减少测试时误操作。
- 图片导出改为 16:9 横屏图，方便电脑查看。
- 导出图坐标图改为正方形。
- 导出图右侧结果精简为 `Rc`、`Rsh`、`R²`。
- 导出图顶部加入导出时间。
- 导出图表格放大，只显示间距与电流值。
- 图片导出优先使用 Flet 原生截图，减少 Android 打包对二进制 Python 包的依赖。
- 历史记录上限提升到 1500 条。

### 修复

- 修复导出图中文显示异常的问题。
- 修复旧版导出图结果项过多、表格过小的问题。

## v0.4 - 旧版

- 基础 TLM 计算。
- 支持输入间距、电流、通道宽度和测试电压。
- 支持历史记录保存与加载。
